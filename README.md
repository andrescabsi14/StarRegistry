A DApp to register stars. Part of the Udacity's Blockchain Developer Nanodegree <br />

## Dependencies

Truffle `v5.1.1 (core: 5.1.1)` <br />
Solidity `v0.5.12 (solc-js)` <br />
Node `v10.15.3` <br />
Web3.js `v1.2.2` <br />
OpenZeppelin `v2.4.0` <br />
